# Changelog

All notable changes to this website are tracked here.

## 2026-05-10

### Added

- Added the v0.3.0 of the static personal website for Diptanil Chaudhuri.
- Moved repository from https://github.com/tech-anarchy/personal_website to https://github.com/dcxforge/diptanil.com
- Added core pages for the site: home, about, work history, research, projects, astrophotography, and contact.
- Added a shared visual system in `assets/css/style.css`, including responsive layouts, cards, timelines, galleries, terminal-style panels, dark mode colors, and mobile navigation styles.
- Added shared JavaScript partials for the site header and footer in `assets/js/partials.js`.
- Added site behavior in `assets/js/index.js`, including dark mode persistence, active navigation highlighting, and mobile menu toggling.
- Added astrophotography gallery content and image assets for deep-sky targets including the Eagle Nebula, Orion Nebula, Rosette Nebula, Crab Nebula, Beehive Cluster, and Pillars of Creation detail.
- Added personal portrait assets, mascot character assets, and light/dark mascot scene banners for the home and contact pages.
- Added draft notes infrastructure with a notes index and article template.
- Added `.gitignore` entries for editor, IDE, and OS metadata.
