# diptanil.com

Personal website for Diptanil Chaudhuri.

Moved repository from https://github.com/tech-anarchy/personal_website to https://github.com/dcxforge/diptanil.com

## Overview

This repository contains a hand-written static website built with HTML, CSS, and small JavaScript helpers. It presents Diptanil's work, research, projects, astrophotography, notes, and contact information in a personal workshop-style design.

The site uses shared header and footer partials, a responsive layout system, light/dark mode support, and custom visual assets including mascot scenes, astrophotography images, and personal portraits.

## Pages

- `index.html` - home page with a short introduction, project direction, and astrophotography highlights.
- `about.html` - personal background, current focus areas, facts, portraits, and mascot overview.
- `work.html` - professional timeline and experience summary.
- `research.html` - research focus areas and publication list.
- `projects.html` - placeholder for current and future project writeups.
- `astrophotography.html` - gallery of deep-sky images and gear/process notes.
- `contact.html` - contact links, reasons to reach out, and email details.
